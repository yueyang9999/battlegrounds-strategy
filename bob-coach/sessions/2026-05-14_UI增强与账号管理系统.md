# 会话记录: UI增强与账号管理系统

**日期**: 2026-05-14
**分支**: main

## 完成内容

### Phase 1: 冻结/刷新视觉反馈增强
- 新增 `@keyframes freeze-glow` 冰蓝呼吸光效冻结提示动画
- 新增 `#freeze-indicator` 酒馆冻结浮动雪花标记（`@keyframes freeze-float`）
- 冻结/解冻/智能刷新 hint-item 边框和颜色增强
- `renderFreezeIndicator()` 函数串联到 `renderAll()`

### Phase 2: 图例说明面板
- `#legend-overlay` modal 面板，5个分区：
  - 卡片标记（核/战/碰/买/卖颜色含义）
  - 决策图标（升本/搜牌/冻结/解冻/技能/法术/饰品/警告）
  - 评级体系（S/A/B/C/D 颜色与含义）
  - 置信度颜色（金色≥70%/蓝色40-69%/灰色<40%）
  - 状态标识（新鲜/稳定/过时攻略标签）
- 齿轮菜单新增入口

### Phase 3: 游戏规则参考面板
- `#rules-overlay` modal 面板，5个分区：
  - 基础买卖规则（购买/出售/刷新/冻结费用）
  - 升本费用表（1本→6本，含每回合-1说明）
  - 卡池数量（7个星级：独特卡牌数/每卡张数/总卡数）
  - 伤害上限规则（回合分段 + 前4无上限）
  - 饰品回合（6/9回合）
- 数据基于 cards.json 实测统计

### Phase 4: 战斗预测设置
- 设置面板新增：显示/隐藏战斗预测条 checkbox
- 模拟次数选择：100/200/500/1000 次
- `computeCombatPrediction()` 使用 `state.combatSimSamples` 替代硬编码200
- `renderCombatPredictor()` 检查 `state.showCombatPredictor` 开关
- 完整串联 openSettings/closeSettings/saveSettings/init

### Phase 5: 数据来源标注 + 免责声明
- 首次协议对话框扩充：数据来源声明（Gamerhub + Firestone）、免责声明（非商业用途）、数据隐私说明
- 关于面板新增数据来源表格和详细免责条款
- CSS: `.agreement-divider`、`.about-source-table` 样式

### Phase 6: 本地用户数据库 + 账号管理系统
- 新建 `modules/UserDataStore.js`：
  - JSON 文件本地存储（原子写入）
  - 用户档案管理（userId/username/email/accountType/privacyLevel）
  - SHA-256 加盐密码哈希
  - 游戏记录 CRUD + 自动统计更新
  - 同步队列管理（离线排队）
  - 数据导出/导入/删除
- 三级数据隐私：纯本地(L0) / 匿名统计(L1) / 完整云同步(L2)
- 10个云端 API 端点预留（当前返回501）
- 账号管理 UI：注册/登录/注销/隐私级别切换/数据导入导出/个人统计面板
- IPC handlers：16个 `user:*` 通道
- preload.js：20个 contextBridge 方法
